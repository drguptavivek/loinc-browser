# LOINC License Note For Agents

## Copyright

LOINC release data is licensed under the [LOINC license](https://loinc.org/kb/license/). This repository must not commit or copy licensed LOINC release zip files, extracted release directories, generated SQLite databases, or generated WAL/SHM files.

Use the MCP tools for narrow search, detail, fit, answer-list, panel, hierarchy, part, and group calls. Do not request or produce bulk dumps of the full release.

When writing outputs for users, include only the minimal LOINC identifiers and term details needed for the task. Do not paste large tables of release data into external systems or source control.

When incorporating extracted LOINC material into products, APIs, data dictionaries, or technical specifications, keep extracted data associated with the relevant LOINC identifier and an official display name, and include appropriate LOINC notice text where required by the license. Check source/copyright metadata before redistributing third-party copyrighted content embedded in LOINC.
